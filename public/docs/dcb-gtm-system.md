# DCB Hospital + SNF GTM System

**Source:** `portmi3-ai/discharge-bridge-media/marketing/go-to-market-system-hospital-and-snf.md`

## Core idea

Turn content and narrative into a repeatable conversion loop:

content -> pipeline -> demos -> pilots

Then expand through network effects:

1 hospital -> multiple SNFs -> better match quality -> faster placement -> more hospitals.

## Hospital target

Primary stakeholders:

- Directors of Case Management
- Throughput leaders
- Discharge operations leaders
- Executive sponsors as required

Entry point:

1. Post infrastructure POV content
2. Send a light DM asking whether it would be helpful to see how placement time can be reduced
3. Offer a short walkthrough

## Pilot offer

- 1 hospital
- 2-3 SNFs
- Limited rollout scope
- Measurable baseline and success criteria

## SNF target

Primary stakeholder:

- Admissions director or census leadership

Strategy:

Sell visibility and inbound consistency, not software.

Message:

Discharge Bridge brings real-time hospital referrals.

## Network effect loop

Once one hospital and five SNFs are active, the local network can create consistent referral volume, improved placement quality, and faster hospital movement.

Expansion path:

local -> region -> state -> national

## Execution cadence

- Monday-Thursday: outbound execution windows
- Friday: pipeline hygiene, messaging review, next-week plan
