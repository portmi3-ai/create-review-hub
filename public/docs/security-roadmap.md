# Security Roadmap

## Positioning

Use HIPAA-aligned and SOC 2 readiness language unless formal certification status is complete.

## Current posture

- Role-based access
- Server-only service role usage
- Tenant-aware workflows
- Audit-oriented routing
- Deterministic workflow controls
- No PHI in investor environments

## Readiness workstream

- Policies
- Access reviews
- Vendor inventory
- Change management
- Incident response
- Logging and monitoring

Pair this roadmap with policies and architecture evidence before formal diligence.
